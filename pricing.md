# Pricing — Revalidation Copilot

## Free
- Price: £0 (one-time)
- AI generations: 1 free reflective account
- Dashboard access: Full (CPD tracking, practice hours, feedback storage, reflective discussion notes)
- Export: Manual copy from dashboard
- Annual subscription: Not required

## Pro
- Price: £12.99/year (billed annually)
- AI generations: Unlimited
- Save and export: Unlimited reflective accounts to PDF
- Features: All dashboard features, voice-to-text reflections, Code theme auto-detection, priority support
- Limits: Single user (NMC-registered nurse or midwife)
- Refund policy: 14-day money-back guarantee
